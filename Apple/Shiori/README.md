# Shiori - AI PDF Reader

専門書・論文を快適に読むためのiPadアプリ

## 主な機能

- 📚 **本棚管理** - PDFを分野ごとに整理
- 🔍 **しおり検索** - 「あれどこだっけ？」をAIで解決
- ✏️ **スマートマーカー** - Apple Pencilでハイライト
- 🔊 **音声読み上げ** - ページを読み上げ
- 🇯🇵 **AI翻訳** - 英語論文を日本語に
- 📄 **論文要約** - 全体を自動要約
