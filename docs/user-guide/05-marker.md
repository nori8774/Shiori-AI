# マーカー機能の使い方

## 概要

Apple Pencilを使って、PDFに直接マーカー（ハイライト）を引くことができます。

紙の教科書に蛍光ペンで線を引くような感覚で、重要な箇所をマークできます。

---

## 基本操作

### マーカーモードをONにする

読書中に **✏️** ボタンをタップ。

ボタンがオレンジ色になり、マーカーモードがONになります。

```
通常モード: ✏️ (グレー)
マーカーモード: ✏️ (オレンジ)
```

### マーカーを引く

1. マーカーモードをONにする
2. Apple Pencilでテキスト上をなぞる
3. 自動的にテキスト行にスナップしてハイライトが適用

> 💡 フリーハンドで引いても、最も近いテキスト行に自動でスナップします

### マーカーモードをOFFにする

もう一度 **✏️** ボタンをタップ。

通常の読書モードに戻ります。

---

## マーカーの色

5色から選べます：

| 色 | おすすめの用途 |
|----|--------------|
| 🟡 黄色 | 重要ポイント |
| 🩷 ピンク | 疑問・要確認 |
| 🔵 青 | 定義・用語 |
| 🟢 緑 | 例・事例 |
| 🟠 オレンジ | 結論・まとめ |

### 色の変更方法

**✏️** ボタンを**長押し**すると、マーカー設定画面が開きます。

```
┌─────────────────────────┐
│ マーカー設定            │
├─────────────────────────┤
│ 色:  🟡 🩷 🔵 🟢 🟠    │
│                         │
│ 太さ: ─  ━  █          │
│       細  中  太        │
└─────────────────────────┘
```

好きな色をタップして選択。

---

## マーカーの太さ

3種類から選べます：

| 太さ | 見た目 | 用途 |
|------|--------|------|
| 細 | アンダーライン風 | さりげなくマーク |
| 中 | 標準ハイライト | 通常使用 |
| 太 | 行全体をカバー | 特に重要な箇所 |

---

## マーカーの削除

### 個別に削除

1. 削除したいマーカーを**長押し**
2. 「削除」をタップ

### ページ内のマーカーを一括削除

マーカー設定画面（✏️長押し）から「このページのマーカーを全削除」を選択。

---

## しおりとの連携

### 自動しおり機能

マーカーを引くと、そのページに**自動的にしおり**が付きます。

しおりのリボンの色は、そのページで最初に引いたマーカーの色になります。

```
🔖（黄色のリボン）← 黄色のマーカーを引いた
🔖（青色のリボン）← 青色のマーカーを引いた
```

### セマンティック検索との連携

マーカーで引いたテキストは、しおり検索の対象になります。

検索結果には、マーカーで引いた箇所も表示されます：

```
p.42  技術書.pdf              85%
ニューラルネットワークの基礎...
✏️ 活性化関数 / 勾配降下法    ← マーカー箇所
```

## 活用例

### 色分けで整理する読書法

**論文を読むとき：**
- 🟡 黄色: 著者の主張・結論
- 🔵 青: 専門用語の定義
- 🟢 緑: 実験結果・データ
- 🩷 ピンク: 後で調べたい箇所
- 🟠 オレンジ: 自分の研究に使えそうな部分

**教科書を読むとき：**
- 🟡 黄色: 重要な公式・定理
- 🔵 青: 用語の定義
- 🟢 緑: 具体例
- 🟠 オレンジ: 章のまとめ

### 復習に活用

1. マーカーを引きながら読書
2. 後日、しおり検索で「まとめ」と検索
3. オレンジでマークした箇所が一覧で見つかる
4. 効率的に復習できる

## Tips

### Apple Pencilの筆圧

筆圧によってマーカーの太さが変わることはありません。
設定で選んだ太さが常に適用されます。

### 指でもマーカーは引ける？

マーカーモードでは指でのタッチも有効ですが、Apple Pencilの方が精度が高くおすすめです。

### マーカーはPDFに保存される？

マーカーはアプリ内のデータとして保存されます。
**元のPDFファイルは変更されません**。

これにより：
- いつでもマーカーを削除できる
- 元のPDFを他のアプリで開いても影響なし

### マーカーのバックアップ

現在、マーカーデータのエクスポート機能はありません。
アプリを削除するとマーカーも消えるのでご注意ください。

---

## トラブルシューティング

### マーカーが反応しない

1. マーカーモードがONになっているか確認（✏️がオレンジ色）
2. Apple Pencilが接続されているか確認
3. アプリを再起動してみる

### マーカーがずれる

テキスト認識の精度によっては、意図した位置とずれることがあります。
その場合は、マーカーを削除して引き直してください。
