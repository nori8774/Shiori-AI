# 本棚機能の使い方

## 概要

本棚機能を使えば、増えていくPDFを分野やプロジェクトごとに整理できます。

---

## システム本棚

アプリには2つのシステム本棚が用意されています：

| 本棚 | 説明 |
|------|------|
| **すべて** | ライブラリの全書籍を表示 |
| **要約** | 論文要約PDFが自動で入る |

> 💡 システム本棚は削除できません

---

## 本棚の作成

### 新しい本棚を作る

1. 本棚タブの右端にある **＋** をタップ
2. 本棚名を入力（例：「機械学習」「プロジェクトA」）
3. 「作成」をタップ

### 本棚名の変更

1. 本棚タブで編集したい本棚を長押し
2. 「名前を変更」を選択
3. 新しい名前を入力

---

## 本を本棚に入れる

### 方法1: インポート時に選択

PDFをインポートすると、本棚選択ダイアログが表示されます。
入れたい本棚を選んでください。

### 方法2: 後から変更

1. ライブラリで本を **長押し**
2. コンテキストメニューから「本棚を変更」をタップ
3. 移動先の本棚を選択

```
┌─────────────────────┐
│  本棚を変更          │
├─────────────────────┤
│  ○ すべて           │
│  ○ 要約            │
│  ● 機械学習   ✓     │  ← 選択中
│  ○ プロジェクトA    │
└─────────────────────┘
```

---

## 本棚の切り替え

画面上部の本棚タブをタップすると、その本棚に入っている本だけが表示されます。

```
[すべて] [要約] [機械学習] [プロジェクトA]
           ↑
        タップで切り替え
```

---

## 本棚の削除

1. 本棚タブで削除したい本棚を長押し
2. 「削除」を選択
3. 確認ダイアログで「削除」をタップ

> ⚠️ **注意**: 本棚を削除しても、中の本は削除されません。
> 本は「すべて」本棚で引き続き見られます。

## Tips

### 本棚を使った効率的な学習

1. **読み始める本** → 専用の本棚を作成
2. **読書中** → しおりとマーカーで重要箇所をマーク
3. **読み終わったら** → 「読了」本棚に移動

### 検索との組み合わせ

しおり検索は全本棚横断で検索されます。
「あの概念、どの本に書いてあったっけ？」という時に便利です。
