# PDF Reader 基本ガイド

## はじめに

PDF Readerは、専門書や論文などのPDFを快適に読むためのiPadアプリです。

「あれ、どこに書いてあったっけ？」という悩みを解決するAI検索機能や、Apple Pencilでのマーカー機能など、学習・研究に特化した機能を搭載しています。

---

## 初回起動

### PDFを追加する

1. アプリを起動すると、空のライブラリ画面が表示されます
2. 右上の **＋** ボタンをタップ
3. ファイルアプリからPDFを選択
4. インポート後、本棚を選択するダイアログが表示されます

### 本の開き方を設定する

PDFを初めて開くとき、開き方向を聞かれます：

- **右開き（縦書き）**: 日本語の小説、漫画など
- **左開き（横書き）**: 英語の論文、技術書など

一度設定した方向は保存され、次回からは自動で開きます。

> 💡 **変更方法**: 本棚で本を長押し → 「開き方向の設定」から変更できます

## 画面の見方

### ライブラリ画面

```
┌────────────────────────────────────────┐
│ ⚙️  ライブラリ           🔍 📝 ➕      │
├────────────────────────────────────────┤
│ [すべて] [論文] [技術書] [＋]          │ ← 本棚タブ
├────────────────────────────────────────┤
│                                        │
│   📕  📗  📘  📙                        │ ← 本の一覧
│                                        │
└────────────────────────────────────────┘
```

- **⚙️**: 設定画面（APIキーの設定など）
- **🔍**: 検索
- **📝**: 読書履歴
- **➕**: PDFを追加

### 読書画面

```
┌────────────────────────────────────────┐
│ ← ライブラリ      🔍 ✏️ 🔖 ✨AI解析 👁 │
├────────────────────────────────────────┤
│                                        │
│                                        │
│            PDFページ                    │
│                                        │
│                                        │
└────────────────────────────────────────┘
```

- **🔍**: しおり検索（タップ）/ PDF内検索（長押し）
- **✏️**: マーカーモード ON/OFF
- **🔖**: しおりを追加/削除
- **✨AI解析**: AI機能メニュー
- **👁**: 表示設定（見開き/単ページ）

---

## ページめくり

### スワイプ操作

- **左スワイプ**: 次のページへ
- **右スワイプ**: 前のページへ

### 見開き表示

1. 右上の **👁** ボタンをタップ
2. 「見開き表示」を選択
3. 2ページ同時に表示されます

> 💡 論文や技術書は見開き表示が読みやすい場合があります

---

## しおり

### しおりを付ける

読書中に **🔖** ボタンをタップするだけ。

しおりを付けると：
- ページ右上にリボンアイコンが表示
- AIが自動でページ内容を要約
- セマンティック検索の対象になる

### しおりを外す

同じ **🔖** ボタンをもう一度タップ。
