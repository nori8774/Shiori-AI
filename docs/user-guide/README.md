## アプリの特徴

### 🎯 こんな人におすすめ

- **研究者・大学院生**: 論文を大量に読む方
- **エンジニア**: 技術書で学習する方
- **学生**: 教科書・参考書で勉強する方

### ✨ 主な機能

| 機能 | 説明 |
|------|------|
| 📚 本棚 | PDFを分野ごとに整理 |
| 🔍 しおり検索 | 自然言語でページを探す |
| ✏️ マーカー | Apple Pencilでハイライト |
| 🔊 音声読み上げ | ページを読み上げ |
| 🇯🇵 翻訳 | 英語論文を日本語に |
| 📄 論文要約 | 全体を自動要約 |

### 🔧 必要なもの

- iPad（iOS 17以上）
- Apple Pencil（マーカー機能を使う場合）
- Google AI Studio APIキー（AI機能を使う場合・無料）

## 補足情報

### APIキーについて

Google Gemini APIには無料枠があり、通常の使用では費用はかかりません。

- 1分あたり15リクエスト
- 1日あたり1,500リクエスト

### プライバシー

- PDFファイルは端末内に保存
- AI解析時のみGoogleサーバーに画像を送信
- APIキーはKeychainで暗号化保存
